# My AI Agent - Built on OpenClaw

A customizable AI agent built by following the [build-your-own-openclaw](https://github.com/czl9707/build-your-own-openclaw) tutorial.

## Custom Skills I Added
- **Shakespeare mode** - Bot speaks in Shakespearean English
- **Pirate mode** - Bot talks like a pirate, arrr!
- **Hinglish mode** - Bot mixes Hindi and English like a true desi
- **Sarcastic mode** - Bot is dramatically unimpressed but still helpful

## How to Run
1. Clone this repo
2. Copy config: cp default_workspace/config.example.yaml default_workspace/config.user.yaml
3. Add your API key to config.user.yaml
4. cd into any step folder and run: uv run my-bot chat
